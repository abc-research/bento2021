# How to start to make this page
[doc](http://jekyllrb-ja.github.io/docs/)

## setup
```
$ gem install jekyll bundler
$ clone git@github.com:abc-research/nurse2021.git
$ cd nurse2021
$ bundle install
```

## run surver
```
$ bundle exec jekyll serve
```
